Система управления лекарствами: категории и лекарства со связью.
Интерфейс использует только понятные пользователю данные (без ID).

# Описание
Программа представляет собой консольное приложение для работы с базой данных с помощью библиотеки psycopg2.
Поддерживает добавление, просмотр, удаление и редактирование сущностей, поиск по ключу. Управление реализуется через интерфейс, не использующий технические значения и суррогатные ключи. Классы таблиц хранятся в tables/ и наследуются от dbtable. В качестве примера созданы таблицы drug_categories и drugs. Подключение осуществляется с учетом файла конфигурации config.yaml.
